ImageData plotting tool?

Plot Over Line

Using the Data Analysis -> Plot Over Line filter, we can sort of dynamically plot a row of an ImageData dataset. Note that to have the Line Source be displayed/pickable, the "PlotOverLineX" object in the Pipeline needs to be selected and the ImageData render window needs to be selected. By holding down "Shift", the user can move the line source, constrained to move only in the Y direction. Also, by setting the line source's Resolution=9 (our DataImage is 5x10), then mousing over the implied points in the plot, we will see values at the exact X (integral) locations. Here's the simple_2.vti file which contains the following matrix:

0.000 3.214 4.924 4.330 1.710 -1.710 -4.330 -4.924 -3.214 0.000
