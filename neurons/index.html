NeuroPy: A tool to analyze neuron firing

The point of this exercise is to explore the possiblity of using an existing open source application (ParaView) for visualizing and analyzing neurons (firing) in cortical networks.

Set background, edge, etc colors and auto-update via Preferences.

Initially, played with (20x20 grid) POLYDATA having both SCALAR and VECTOR data.

Switched to simpler STRUCTURED_POINTS (Image) grid with just SCALAR data (0/1; firing).
