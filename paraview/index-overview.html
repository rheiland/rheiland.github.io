    The instructions for "verifying that parallel rendering is working" from the wiki is not quite correct (for v. 3.5.0 Dev). Instead, do: Paraview->Prefs: RenderView->Server, then check the Remote Render Threshold box and move slider to the left (0 Mbytes):

    When we then interact with the rendering, it appears pixelated while moving, but renders smooth when we stop.

    It seems that the same effect (pixelation) does not apply to Overview.
