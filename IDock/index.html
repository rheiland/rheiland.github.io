IDock - Immersive Molecular Docker

While working at NCSA, I worked on an interesting molecular docking application. The PI for the project was Dr. Ruth Pachter at the AFRL. It was a legacy code that needed upgrading and porting to an immersive visualization environment. Later, we also added haptics and voice commands to the applicaton.

Demo'ing the full application at Illinois TechDay 2001 at the State Capitol in Springfield.
