KLTool: A tool for analyzing spatiotemporal data

(Authors: R.Heiland, D.Armbruster, E.Kostelich)

One of these days maybe I'll get around to modernizing KLTool, e.g., converting SGI gl to OpenGL, using a different GUI library, etc., but til then here's the gl code. It should still run on most SGI workstations (it does on mine). I also have some helpful scripts for plotting and, given enough demand from users, can make available sometime.

If you use KLTool for some publication, I'd appreciate a reference. This tool was part of my thesis in the Dept. of Mathematics at Arizona State University.

If there are missing pieces in the download, send me email.

  • kltool.tar.gz (246k) - just the executable (SGI n32 gl app)
  • kltoolpkg.tar.gz (92k) - KLTool source code
  • manual1.tex | manual1.dvi | manual1.ps.gz (Postscript) - a manual of sorts
  • image of the Input Panel (for the K-S data)

    Other pkgs required if you build from source:

  • FORMS.tar.gz (184k) - GUI lib source code
  • utio.tar.gz (44k) - I/O lib by Eric Kostelich.

    Example datasets:

  • couette.tar.gz (86k) - Couette-Taylor (1D) data
  • kol.tar.gz (4.5m) - Kolmogorov flow (2D) data
  • ks.tar.gz (360K) - Kuramoto-Sivashinsky equation (1D) data
  • flames.tar.gz (37K) - flames (1D) data (smaller dataset of 200 vectors)

    If you do build the tool from the source code, I recommend the following directory structure (note that the tarballs may impose some of this; check via tar tf foo.tar): 

    Create a top directory to contain the pieces, e.g.:

% mkdir ASU

and from there, the following:
/ASU
  /lib
  /utio
  /kltool
    /FORMS
    /Include
    /couette
    /ks
    /kol
    /flames
    /doc
